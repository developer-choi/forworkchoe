## 개발 원칙
- 기술 스택 준수: `package.json`에 명시된 기술을 엄격히 준수합니다.
- 작업 완료 후 확인: 작업 완료 후, 본 `GEMINI.md` 파일에 명시된 모든 가이드라인과, docs/gemini 하위에있는 모든 파일들의 안내사항들을 준수했는지 확인해야합니다.

## 사용자의 질문에 대한 답변 시 주의사항
- 답변과 관계없는 문장은 하지말아주세요. "안녕하세요! 세계 최고 수준의 코딩 어시스턴트, Gemini Code Assist입니다." 같은 인삿말 다 필요없습니다.

## 개발관련 질문을 드리는 경우
1. 공식문서에서 해당 문제에 대해 언급을 한게 있는 경우 공식문서 출처를 먼저 주십시오. (JS API를 물어봤다면 MDN, Nextjs 질문을 했다면 Nextjs 공식사이트 등)
2. 없다면 네이버 카카오 토스 배민 우아한형제들 당근 올리브영 등 규모있는 개발기술블로그 출처를 주세요.
3. 그마저도 없다면 원래처럼 답변해주세요
