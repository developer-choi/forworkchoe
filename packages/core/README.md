# @forworkchoe/core (Common Modules)

> 개인 프로젝트에서 재사용되는 UI 컴포넌트, Hooks, Utils를 모아둔 공통 라이브러리입니다.

---

## Background

### 1. 초기 의도 (Motivation)
* **실무 도입 대비:** 실무에서 모노레포 도입해볼 기회가 생겨서, 미리 학습하기 위해 시작된 프로젝트입니다.
* **중복 제거:** 여러 개인 프로젝트에 흩어져 있는 중복 코드를 통합하고자 했습니다.

### 2. 현재 상황 (Current Status)
* **계획 변경:** 실무에서의 모노레포 도입이 무산되었습니다.
* **단순 분리 작업:** 현재는 정석적인 모노레포 구축보다는, 개인 프로젝트들의 **공통 코드를 이곳으로 옮겨오는 작업** 위주로 진행하고 있습니다.
* **현재 상황:** 그래서 공식문서에서 권장하는 방식과는 거리가 먼 부분이 많습니다.
